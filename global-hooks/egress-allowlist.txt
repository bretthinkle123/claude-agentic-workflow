# (Layer 2 proxy) and/or flagged (Layer 3 egress-check.sh). One host per line; `#` comments and
# blank lines ignored; a leading `.` means "this host and any subdomain" (e.g. `.docker.io`).

# representative pipeline with egress LOGGED and blocking OFF, then reconcile this list against
# the observed hosts) before trusting it as the enforced ACL. Over-blocking breaks a scanner;
# under-blocking misses a real path — so measure, don't guess.
